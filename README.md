# STARMIND

Plataforma educacional gamificada desenvolvida para auxiliar no processo de aprendizagem infantil.

## Objetivo
Desenvolver uma plataforma interativa capaz de tornar o aprendizado mais atrativo por meio da gamificação.

## Funcionalidades
- Sistema de pontos e medalhas
- Desafios educativos
- Feedback imediato
- Conteúdos educativos
- Interface intuitiva

## Tecnologias utilizadas
- Java
- HTML/CSS
- Figma
- Scrum
